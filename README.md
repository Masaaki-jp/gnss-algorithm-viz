# GNSS Algorithm Visualization with Gemini Deep Research

Google の次世代リサーチ機能 **Gemini Deep Research** を活用し、GNSS（全球測位衛星システム）の高度な測位アルゴリズムと精度メカニズムを解析・可視化したプロジェクトです。

## 🌐 Live Demo
GitHub Pages で公開されているインタラクティブな可視化ページはこちら：
[https://masaaki-jp.github.io/gnss-algorithm-viz/](https://masaaki-jp.github.io/gnss-algorithm-viz/)

## 📝 プロジェクト概要
本プロジェクトは、複雑な専門知識を「AIによる深い洞察」と「ウェブ技術による視覚化」でいかに分かりやすく解体できるかという **Survival DX**（低スペックな環境からクラウドとAIを駆使して最大のアウトプットを出す手法）の実践例です。

### 主な解析・可視化内容
- **三辺測量と第4の次元:** 衛星3機による座標特定と、4機目による時刻補正の原理。
- **誤差要因の分析:** 電離層遅延やマルチパスが精度に与える影響の定量的ビジュアル。
- **測位精度の進化:** 単独測位から PPP、RTK に至る精度の変遷（メートル級からセンチメートル級へ）。
- **マルチGNSS:** GPS, Galileo, BeiDou など、世界の衛星群の比較。

## 🛠 テクニカルスタック
- **Analysis:** Gemini Deep Research (Next-generation AI Model)
- **Frontend:** HTML5, CSS3 (Tailwind CSS)
- **Visualization:** [Chart.js](https://www.chartjs.org/)
- **Hosting:** GitHub Pages

## 💡 Survival DX 視点での意義
このプロジェクトは、安価な Chromebook のブラウザ上のみで完結しています。
1. **AI による重厚なリサーチ:** 数十件の専門資料を読み解く代わりに AI が数分で論理を構築。
2. **コードの高速生成:** 複雑な Chart.js や CSS アニメーションを AI との協働で実装。
3. **サーバレス公開:** インフラ構築不要で GitHub Pages による即時デプロイ。

高価なワークステーションや専門ソフトがなくとも、適切なプロンプトとクラウド・エンジニアリングがあれば、プロフェッショナルな技術レポートを作成できることを証明しています。
